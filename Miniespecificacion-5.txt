
Generar un solución que permita calcular y mostrar el pago mensual de un préstamo de 1 año de plazo. Se debe ingresar el monto del préstamo y el interés mensual a cobrar.

Algoritmo:

1. Inicio 
2. Se pide ingresar el monto del préstamo
3. Se pide ingresar el interés mensual a cobrar
4. Se divide el monto del préstamo para 12 meses y se suma el interés mensual a cobrar
5. Se muestra en pantalla el pago mensual de un préstamo.
6. Fin


---------- Miniespecificacion o Pseudocódigo 
---------- Opción 1

1. Inicio
//declaro variables

2. monto_préstamo, d[0, n]
3. interés_mensual_cobrar, d [0, n]
4. pago_mensual 

// pido datos por teclado - entradas

5. Escribir"Ingresar el monto del préstamo"
6. Leer monto_préstamo
7. Escribir "Ingrese el interés mensual a cobrar"
8. Leer interés_mensual_cobrar

// proceso

9. pago_mensual = monto_préstamo / 12 + interés_mensual_cobrar

// salida

10. Ingresar " El pago mensual del préstamo es: " + pago_mensual
11. Fin
